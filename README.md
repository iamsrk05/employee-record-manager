
# Employee Record Management System

A beginner-level employee record management and reporting project developed using Python, Excel, CSV, SQL, and GitHub.

## Features
- Employee data management
- Excel spreadsheet handling
- SQL database integration
- Weekly export reports
- Daily production reports
- Data analysis and summaries
- Technical documentation

## Technologies Used
- Python
- Microsoft Excel
- SQL
- Pandas
- Git & GitHub
